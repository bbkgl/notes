# 78. Subsets

---
100%，开始想到了这个规律，但是翻评论一看，已经有大佬说了。就是先遍历给定数组中所有元素，然后遍历之前生成的答案序列，对每个子序列加上该元素，组成新的子序列，再加入到答案总序列中。。。以此类推

```cpp
class Solution {
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> ans;
        vector<int> temp;
        ans.push_back(temp);
        for (int it : nums) {
            int n = ans.size();
            for (int i = 0; i < n; i++) {
                temp = ans[i];
                temp.push_back(it);
                ans.push_back(temp);
            }
        }
        return ans;
    }
};
```
